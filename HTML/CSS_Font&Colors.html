Let's get started with Colors

Color Can Be Represented In:

  1. Color Keywords
  2. Hex Color Code
  3. RGB
  4. RGBA
  5. HSL
Examples:
  1. KeyWord Color -- pink
  2. Hex Code -- #0f35df9e;
  3. RGB Values -- rgb(204, 13, 96)
  4. RGBA Values -- rgba(52, 57, 165, 0.705);
  5. HSL Values -- hsla(288, 83%, 47%, 0.469)
